## What it is

Hire is pre-priced transactional knowledge work delivered by on-demand AI agents. Buy a deliverable like buying from a vending machine: pick the SKU, pay, agent runs locally on Apple Silicon, deliverable arrives. No proposal calls. No scope creep. No consulting hourly billing.

Two-tier audience built into the same product: solo founders buying small deliverables on impulse, and enterprise CISOs and procurement teams evaluating the compliance package and procurement-defense layer (NCC Group source escrow, source-cessation trigger, hardware EOL roadmap).

## Stack

Built with TypeScript, Node, Stripe, local MLX inference for the agents that run the SKUs. Procurement-grade defense layer: NCC Group source escrow (cessation trigger releases source under MIT to all paying customers), Apple Silicon hardware EOL roadmap, structured 12-month founder license cohort.
